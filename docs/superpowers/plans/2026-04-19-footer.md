# Footer Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a static half-screen footer with the Hydrogen font wordmark left and structured nav/contact content right, on a dark ink + dot-grid background.

**Architecture:** Three isolated changes — wire the Hydrogen font into the Next.js font system, create the Footer component, register it in the page. No animations; no client-side hooks needed.

**Tech Stack:** Next.js (next/font/local), Tailwind CSS v4 (`@theme` tokens), React (server component)

**Files:**
